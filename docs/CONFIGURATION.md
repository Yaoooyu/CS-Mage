# Configuration

`configs/msa/mage_fusion_formal_v2.yaml` is the formal-v2 record. It uses truncated Mixup, not reflection Mixup. `configs/msa/ablations/` records the actual formal-v2 ablation semantics. All output paths are placeholders and must be supplied by the user.

`configs/asr/` records fixed-language Chinese inference/adaptation settings. It is a configuration record, not a one-command launcher.
