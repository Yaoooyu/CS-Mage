# Configuration

`configs/msa/mage_fusion.yaml` is the MAGE-Fusion configuration record. It uses truncated Mixup, not reflection Mixup. `configs/msa/ablations/` records the MAGE-Fusion ablation semantics. All output paths are placeholders and must be supplied by the user.

`configs/asr/` records fixed-language Chinese inference/adaptation settings and their corresponding entrypoints.
